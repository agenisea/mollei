Thanks for your interest in contributing to Mollei! This document outlines how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Install dependencies with `pnpm install`
4. Run `pnpm dev` to start the development server

## Development Workflow

```bash
# Start development server
pnpm dev

# Run linter
pnpm lint

# Build for production
pnpm build
```

## How to Contribute

### Reporting Bugs

- Check existing issues first to avoid duplicates
- Use a clear, descriptive title
- Include steps to reproduce the issue
- Describe expected vs actual behavior

### Suggesting Features

- Open an issue describing the feature
- Explain the use case and why it would be valuable
- Be open to discussion about implementation approaches

### Pull Requests

1. Create a branch from `main` for your changes
2. Make your changes with clear, focused commits
3. Ensure `pnpm lint` and `pnpm build` pass
4. Open a PR with a clear description of changes
5. Link any related issues

## Code Style

- Use TypeScript for all new code
- Follow existing patterns in the codebase
- Keep components focused and single-purpose
- Use meaningful variable and function names

## Areas for Contribution

- **Research**: Emotional intelligence patterns, ethical AI frameworks
- **Documentation**: Tutorials, examples, architecture decisions
- **UI/UX**: Accessibility improvements, design refinements
- **Community**: Discussions, feedback, use case exploration

## Ethical Guidelines

Please review our [STEWARDSHIP.md](./STEWARDSHIP.md) before contributing. Mollei is committed to building emotionally intelligent AI that respects human dignity and autonomy.
